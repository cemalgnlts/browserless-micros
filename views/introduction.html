Introduction

Browserless Micros offers you many ready-to-use micros for free. The Puppetter library and browserless.io enable browser automation for everyone. You don't need any server or installation to use micros, just your API key from your browserless.io account.

If you don't use your own API key when testing micros, you'll use our free key that we offer to everyone, which can quickly fill up the quota. With browserless.io you can quickly get your own API key for free and without a credit card.

Using the API Key

If you're using a micro on the page, you can use it by adding your API key from the top right. Your API key is deleted after the browser is closed and you need to re-enter it each time.

If you are running the micro by sending a request, add x-api-key to the header.
